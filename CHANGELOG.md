## [Unreleased]

### Added
- Initial public release.
- Reverse-connect agent over WSS with auto-reconnect.
- Web panel with live metrics and PTY-based shell tabs (xterm.js).
- Per-agent enrollment: server-issued secret bound to the agent uuid.
- Cookie-based panel sessions with HMAC signing tied to the panel password.
- IP-based throttling and lockout on login failures.
- One-line server bootstrap (`deploy/setup.sh`) and one-line agent install
  (`rclient-agent install --connect ...`).
- Three TLS modes selectable at setup: Cloudflare DNS-01, HTTP-01,
  bring-your-own-certificate.
- systemd and OpenRC service units, generated by `rclient-agent install`.
- "Add device" button in the panel sidebar that hands you a ready-made
  install command.
